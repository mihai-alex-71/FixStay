How FixStay Works

From issue to resolution in four simple steps — fully tracked and transparent.

01
📸

Host Reports Issue

Upload photos, add description, set category and priority level for the problem.

02
🔍

Provider Finds Task

Verified service providers browse available jobs filtered by location, type, and urgency.

03
⚙️

Work Gets Done

Provider accepts, updates status to In Progress, and uploads completion proof when done.

04
⭐

Host Reviews

Host confirms completion and leaves a rating — building trust across the platform.

User Roles

Built for Everyone in the Chain

Three distinct roles, each with a tailored experience designed for their workflow.

🏠

Host / Property Owner

  • Add and manage rental properties
  • Create maintenance requests with photos
  • Track task status in real time
  • Receive notifications on updates
  • Rate and review service providers
  • View full maintenance history
🔧

Service Provider

  • Create a verified service profile
  • Browse tasks by location & type
  • Accept or decline tasks
  • Update status: In Progress / Done
  • Upload completion proof & notes
  • Build reputation through ratings
🛡️

Administrator

  • Manage all users and roles
  • Verify service providers
  • Moderate tasks and reviews
  • View platform analytics
  • Monitor response times
  • Track provider performance
